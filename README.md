## 🚀 Getting Started

To explore the portfolio locally:

1. **Clone the Repo**:
   ```bash
   git clone https://github.com/basseyriman/portfolio-website.git
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   ```bash
   npm run dev
   ```

4. **Visit the Site**: Open [http://localhost:3000](http://localhost:3000)
